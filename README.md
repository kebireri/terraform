This repository documents work carried out using Terraform as part of Infrastructure as Code (IaC) practices. It includes selected learning exercises and implemented projects, providing a single location to record progress, maintain examples, and develop infrastructure configurations over time.

## Repository Structure
```
Terraform Repository
│
├── projects/
│     ├── <project-1>/
│     ├── <project-2>/
│     └── ...
│
├── scripts/
│     ├── <script-1>
│     ├── <script-2>
│     └── ...
│
└── README.md
```
---
**projects/**  
Contains individual Terraform projects. Each directory is intended to hold its own configuration files and any related modules.

**scripts/**  
Contains auxiliary scripts that support the Terraform projects without holding configuration logic.
